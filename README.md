ansible-grafana2
================

An Ansible role for grafana v2 on RHEL/CentOS.

Configuration
-------------
See [`defaults/main.yml`](https://github.com/hmmbug/ansible-grafana2/blob/master/defaults/main.yml) for configuration options.

Example Playbook
----------------

    - hosts: servers
      roles:
         - hmmbug.grafana2
